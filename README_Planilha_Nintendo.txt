
README - PLANILHA DE ASSINATURAS NINTENDO SWITCH
================================================

📁 Arquivo: Planilha Assinatura Nintendo.xlsx

Esta planilha foi desenvolvida para registrar, visualizar e analisar informações sobre assinaturas do serviço Nintendo Switch Online, incluindo planos, modelos de console e dados financeiros agregados. Abaixo segue a explicação de cada aba e instruções de uso.

─────────────────────────────────────────────────────────────
1. 🟥 Aba: A̳ssets
─────────────────────────────────────────────────────────────
Contém a paleta de cores utilizada na planilha, com foco nas cores oficiais da marca Nintendo:
- Nintendo Red:     #E60012
- Nintendo White:   #FFFFFF
- Nintendo Dark Gray: #2D2D2D

Essa aba serve como base visual para manter a identidade visual consistente nas demais abas e gráficos.

─────────────────────────────────────────────────────────────
2. 🟦 Aba: B̳ases
─────────────────────────────────────────────────────────────
Base de dados principal da planilha. Contém os registros individuais de assinantes com as seguintes colunas:

- Subscriber ID: Identificação única do assinante
- Name: Nome da pessoa
- Modelo do Console: Versão do Nintendo Switch utilizado
- Planos: Tipo de plano contratado (Basic, Family Plan, Expansion Pack)
- Start Date: Data de início da assinatura
- Auto Renewal: Indica se a renovação automática está ativada
- Subscription Price: Valor do plano contratado
- Assinatura: Frequência da assinatura (Monthly, Quarterly, Annual)
- Coupon Value: Valor de desconto aplicado
- Total Value: Valor final após descontos

✔️ Use essa aba para inserir novos assinantes ou atualizar informações existentes.

─────────────────────────────────────────────────────────────
3. 📊 Aba: Calculos
─────────────────────────────────────────────────────────────
Contém consolidações e análises dos dados da aba "B̳ases", como:

- Total de valores por tipo de plano
- Segmentações por tipo de assinatura (mensal, anual)
- Tabelas dinâmicas para facilitar a análise

✔️ Recalcule os dados sempre que atualizar a aba "B̳ases".

─────────────────────────────────────────────────────────────
4. 📈 Aba: Dashboard
─────────────────────────────────────────────────────────────
Painel visual com indicadores e dados consolidados da planilha, como:

- Título temático personalizado (ex: "Nintendo Online Subscription Report")
- Dados vinculados dinamicamente à aba "Calculos"
- Design temático com a paleta de cores da Nintendo

✔️ Essa aba é apenas para visualização. Evite editar diretamente os valores, pois eles são puxados das outras abas.

─────────────────────────────────────────────────────────────
✅ Recomendações de Uso:
─────────────────────────────────────────────────────────────
- Sempre use os botões de atualização de Tabela Dinâmica (caso ativados).
- Mantenha a consistência de preenchimento nas colunas da aba "B̳ases".
- Utilize filtros para facilitar análises específicas (ex: apenas planos anuais).
- Cores e logos foram adaptados ao estilo Nintendo para fins visuais.
